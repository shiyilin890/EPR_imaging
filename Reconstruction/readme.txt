How to use:
Step1: install Python (I use version 3.6.6 and above)
https://www.python.org/downloads/  

Step2: install pip
https://pip.pypa.io/en/stable/installing/  

Step3: install library
e.g on Windows, in terminal, type in (arbitrary order)
pip install matplotlib
pip install numpy
pip install scipy
pip install tkinter

Step4: download and unpack; in terminal go to your fold then run:
cd ***(address, e.g. C:/user/folder...)
python rs1spectrum.py (or other files)
